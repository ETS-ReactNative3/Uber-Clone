
# Uber Clone 
A  Uber clone mobile app built using the latest react native and reduxJS.

## Technologies
- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [ReduxJS/ToolKit](https://redux-toolkit.js.org/)
- [TailWind CSS - native](https://docs.nativescript.org/plugins/tailwindcss.html#usage)
- [React Navigation](https://reactnavigation.org/)
- [React Native Elements](https://reactnativeelements.com/)
- [Android Studio](https://developer.android.com/studio/)
- [IOS Emulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Google Maps](https://developers.google.com/maps/apis-by-platform)
- [Google Directions API](https://developers.google.com/maps/documentation/directions/overview)
- [Google Places API](https://developers.google.com/maps/documentation/places/web-service/overview)
- [Google Distance Matrix API](https://developers.google.com/maps/documentation/distance-matrix/start)
